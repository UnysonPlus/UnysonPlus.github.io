# Pin

Sticks an element in place while the page scrolls past it — perfect for a heading or panel that should hold while the next section slides up underneath.

<img src="/img/animations/pin-options.png" alt="Pin — Scroll Motion options panel" width="840" />

## Options

| Option | Notes |
| --- | --- |
| **Pin length** | How long it stays pinned, in % of screen height (100 = one full screen of scrolling) |
| **Run on mobile** | Off by default |

## Set it up

1. Open the **Section** you want to pin → **Animations** tab → **Scroll Motion (GSAP)**.
2. Set **Scroll Effect** to **Pin**.
3. Set **Pin length** (100 = pinned for one screenful of scrolling).
4. Make sure there's content **after** it to scroll over.
5. **Save** — the section sticks, then releases.
