# Snippets

Build reusable Page Builder content once, then embed it anywhere with a shortcode. Great for
repeated blocks — CTAs, contact details, promo banners — that should stay consistent and be
editable from a single place.

## Usage

1. Activate **Snippets** from **Unyson+ → Extensions**.
2. Create a **Snippet** and build its content with the Page Builder.
3. Embed it anywhere with:

```text
[snippet id="…"]
```

Edit the snippet once and every place that embeds it updates automatically.
