---
title: "Hidden"
sidebar_position: 32
---


Simple hidden input.

```php
array(
    'type'  => 'hidden',
    'value' => 'default value',
    'attr'  => array( 'class' => 'custom-class', 'data-foo' => 'bar' ),
)
```

:::tip
The hidden input is not visible, so parameters like `label`, `desc` and `help` have no sense here.
:::
