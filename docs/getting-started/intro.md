# Introduction

**Unyson+** is a free, drag &amp; drop framework for **WordPress** that makes it fast and
easy to build premium themes. It is a modernized continuation of the Unyson framework,
updated for **PHP 7.4+** and current WordPress, and maintained by
[Unyson+](https://github.com/UnysonPlus).

:::note
This manual assumes a working knowledge of WordPress. If you're new to WordPress, start
with the [WordPress documentation](https://wordpress.org/documentation/).
:::

## What you get

- **Page Builder** — a visual drag &amp; drop builder with ready-made shortcodes.
- **Options Framework** — dozens of option types for theme settings and meta boxes.
- **Extensions** — modular features (Portfolio, Sidebars, Breadcrumbs, SEO, Forms,
  Custom Fields, Post Types, and more) you install only when you need them.
- **GitHub auto-updates** — the plugin, theme, and extensions update directly from GitHub.

## How the pieces fit together

| Piece | What it is |
| --- | --- |
| **Unyson+ plugin** | The core framework. Install it like any WordPress plugin. |
| **Unyson+ Theme** | A Bootstrap 5 parent theme built for the framework. |
| **Extensions** | Optional modules you activate from **Unyson+ → Extensions**. |

## Minimum requirements

- WordPress 6.0 or greater
- PHP 7.4 or greater
- MySQL 5.6+ / MariaDB 10.1+
