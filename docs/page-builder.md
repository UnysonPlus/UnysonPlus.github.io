# Page Builder

The Page Builder lets you compose pages visually from **shortcodes**, using drag &amp; drop —
no code required.

:::note Work in progress
This page is a starting point. Expand it with your own screenshots and walkthroughs.
:::

## Enabling the builder

The Page Builder is part of the **WordPress Shortcodes** / **Page Builder** extension.
Activate it from **Unyson+ → Extensions**, then edit any page and switch the editor to the
builder.

## Basic workflow

1. Open a page in the editor.
2. Drag a **shortcode** (e.g. a section, column, or content block) onto the canvas.
3. Configure it through its options panel.
4. Save — the layout renders on the front end.

## Reusable content

The **Snippets** extension lets you save builder layouts and embed them anywhere with:

```text
[snippet id="..."]
```
