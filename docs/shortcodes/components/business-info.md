# Business Info

Opening hours (with a live Open/Closed status) plus contact details — address, phone, email and links — in several layouts. Tabs: **Content**, **Design**, **Styling**, **Animations**, **Advanced**.

## Content

- **Business Name** — optional heading.
- **Opening Hours** — a repeatable list, pre-filled Mon–Sun; each day has:
  - **Day** — Monday … Sunday.
  - **Closed** — Yes / No (default No).
  - **Opens (HH:MM, 24h)** — opening time, default `09:00`.
  - **Closes (HH:MM, 24h)** — closing time, default `17:00`.
  - **Note** — optional, e.g. "By appointment".
- **Show Open / Closed Status** — Yes (default) / No — computed live from the site timezone and today's hours.
- **Time Format** — `12-hour (5:00 PM)` (default) or `24-hour (17:00)`.
- **Address** — multi-line address.
- **Phone** — phone number.
- **Email** — email address.
- **Website URL** — website link.
- **Map / Directions URL** — map / directions link.

## Design

- **Layout** — image-picker (registry-driven); default `Card`.
- **Highlight Today** — Yes (default) / No — emphasize today's row in the hours list.

## Styling

- **Accent** — accent color.
- **Card Background** — background of the card.
- **Text Color** — body text color.
- **Font Size Preset** — named base text size.
- **Margin & Padding** — spacing around the element.

## Animations

Standard entrance-animation controls (effect, duration, delay, offset).

## Advanced

Custom CSS class/ID, responsive visibility toggles, custom attributes, and margin/padding.
