# Post Carousel

A touch-friendly slider of posts (or any public post type) — image, title, excerpt, meta and a read-more — in several card designs. Tabs: **Query**, **Design**, **Styling**, **Animations**, **Advanced**.

## Query

<img src="/img/shortcodes/post-carousel-query.png" alt="Post Carousel options panel — Query tab" width="1200" />

- **Source** — the post type to pull from (any public post type; default `post`).
- **Taxonomy** — the taxonomy to filter by, e.g. `category`, `post_tag`, or a custom taxonomy (default `category`).
- **Terms** — comma-separated term slugs to include; blank = all.
- **Number of Posts** — how many posts to query (default `9`).
- **Order By** — `Date` (default), `Title`, `Menu order`, `Random`, or `Comment count`.
- **Order** — `Descending` (default) or `Ascending`.

## Design

<img src="/img/shortcodes/post-carousel-design.png" alt="Post Carousel options panel — Design tab" width="1200" />

- **Card Design** — image-picker of card layouts (e.g. `standard`, default).
- **Image Ratio** — `Original`, `16:9` (default), `4:3`, `1:1`, or `3:4 (portrait)`.
- **Show Excerpt** — `Yes` (default) or `No`.
- **Excerpt Words** — number of words in the excerpt (default `18`).
- **Show Date** — `Yes` (default) or `No`.
- **Show Author / Category** — `Yes` or `No` (default).
- **Read More Text** — link text (default "Read More"); blank to hide the read-more link.
- **Slides per View** — `1`, `2`, `3` (default), or `4`.
- **Gap** — spacing between slides (default `4`).
- **Autoplay** — `Yes` or `No` (default).
- **Loop** — `Yes` (default) or `No`.
- **Arrows** — `Yes` (default) or `No`.
- **Dots** — `Yes` (default) or `No`.

## Styling

<img src="/img/shortcodes/post-carousel-styling.png" alt="Post Carousel options panel — Styling tab" width="1200" />

- **Image Style** — a live-preview picker of the reusable **Image Style presets** (crop ratio, corner radius, shape mask, filter, scrim), managed in **Theme Settings → Components → Image Styles**.
- **Accent (links / meta)** — accent color for links and meta.
- **Card Background** — card background color.
- **Title Color** — color of the post title.
- **Text Color** — color of the excerpt / body text.
- **Font Size** — font-size preset for the cards.
- **Margin & Padding** — spacing control for the block.

## Animations

Standard entrance-animation controls (effect, duration, delay, offset).

## Advanced

Custom CSS class/ID, responsive visibility toggles, custom attributes, and margin/padding.
