# Content Elements

The everyday building blocks — text, buttons, media, posts, and more. Drag any of these from
the **Content Elements** tab of the Page Builder.

> Most elements also expose **Background Color**, **Text Color**, and **Margin & Padding**
> for spacing/appearance. Those are omitted from the per-element lists below to keep them short.

## Accordion

Collapsible accordion panels.

**Key options:** Accordion Items (Title, Content, Open by Default), Title Tag, Icon Style,
Icon Position, Custom open/closed-state image or text, Item Numbering, Start Number, Item
Spacing, Title Alignment, Collapsible, Multiple Open, URL Hash Deep-Linking, Show
Expand/Collapse All buttons, and per-state title/content/icon colors.

## Button

A styled, linkable button.

**Key options:** Button Label, Button Link, Open in New Window, Button Icon, Icon Position,
Button Style, Button Size, Button Width (or Custom Width), Alignment, Button State, Hover
Animation.

## Calendar

A date/events calendar.

**Key options:** Population Method, Calendar Type, Start Week On, plus heading and navigation
button colors.

## Call To Action

A title + content + button block to drive an action.

**Key options:** Title, Content, Button Label, Button Link, Open in New Window, Title Color,
Content Color.

## Code Block

Insert raw HTML, CSS, or JavaScript.

**Key options:** Code.

## Divider

A horizontal separator, optionally with text or an icon.

**Key options:** Ruler Type, Line Design, Add Element (text/icon), Divider Text, Select Icon,
Content Alignment, Height, Margin Top/Bottom, Width (%), Line Color, Icon Color, Divider Text
Color.

## Icon

A single icon with an optional title.

**Key options:** Icon, Title, Title Color, Icon Color.

## Icon Box

An icon paired with a title and content — great for feature grids.

**Key options:** Icon (or Custom Emoji/SVG), Title, Title HTML Tag, Content, Icon Position,
Icon Badge, Icon/Title/Content Alignment, Stack on Mobile, Box Link URL, Open in New Tab, Link
Rel Attribute, plus title/content/icon/badge colors.

## Image Content

An image alongside text in a responsive two-column layout.

**Key options:** Image, Content, Image Alt Text, Image Link, Open in New Window, Layout,
Column Ratio, Vertical Alignment, Gap, Mobile Stacking Order, Image Fit, Image Border Radius,
Image Shadow.

## Map

An interactive map with multiple engines/styles.

**Key options:** Population Method, Map Engine, Map Style (CARTO / Stadia-Stamen /
Thunderforest / MapTiler / Google), the matching API keys, Map Type, Map Height, Disable zoom
on scroll.

## Notification

A notification / alert box.

**Key options:** Message, Custom Label, Type, Border Style, Icon (or Custom Emoji/SVG), Layout
Style, Dismissible, Auto-dismiss Seconds, plus label/message/icon colors.

## Posts

A powerful query-and-display element for posts — grid, list, masonry, slider, overlay, or hero
layouts with full element-positioning control.

**Query options:** Post Type, Filter by Taxonomy/Terms, Term Relation, Include/Exclude IDs,
Limit to Authors, Date Range, Posts per Page, Offset, Order By (incl. Custom Field Key), Order
Direction, Exclude Current Post, Sticky Post Handling.

**Layout options:** Layout Mode, Card Style, Image/Content Width Ratio, Desktop/Tablet/Mobile
Columns, Column/Row Gap, Card Padding, Equal Card Heights, Image Size & Crop Ratio, Fallback
Image, Featured Post Treatment, Card Text Alignment, draggable Element Order.

**Content & meta:** Title HTML Tag, Categories/Taxonomy chips, Meta Bar items & layout, Date
Format, Excerpt Source/Length/Suffix, Read-More style & text.

**Navigation & performance:** Pagination Type/Position/Alignment, AJAX Filter Bar, slider
controls (arrows, dots, autoplay, loop), Transient Caching with a configurable lifespan, and
an Empty-State Message.

## Special Heading

An overline + title + subtitle heading block.

**Key options:** Overline, Title, Subtitle, Title Tag, per-part Alignment, Overline Uppercase,
Overline Marker & Position, Element Spacing, Heading Max Width, Title Display Size, Subtitle
Font Size/Max Width, per-part colors and CSS classes.

## Table

A data table with optional interactivity.

**Key options:** Table Preset, Frame (Border Preset), Alternating Row Colors, Row Hover
Highlight, Bordered, Compact, Sticky Header, Caption & Position, Sorting, Search/Filter,
Pagination, Rows Per Page, Rows-Per-Page Selector, Info Line.

## Tabs

Tabbed content panels.

**Key options:** Tabs (Title, Content), Active Tab, Tab Style, Justified Tabs, Tab Alignment,
Tabs Orientation, Fade Animation, plus tab title/content colors.

## Team Member

A team member card.

**Key options:** Image, Name, Job Title, Description.

## Testimonials

Testimonials in a grid or carousel.

**Key options:** Title, Testimonials (Quote, Image, Name, Position, Website Name/Link, Rating),
Layout Type, Grid Columns, Gutter Size, Text Align, Container, Items per Slide, Card Style,
Avatar Position/Shape/Size, Show Rating Stars, Autoplay & Interval, Pause on Hover, Prev/Next &
Indicators, Indicator Style, Wrap Around, plus per-part colors.

## Text Block

A rich-text (WYSIWYG) block.

**Key options:** Content.

## Text Expander

An inline read-more / reveal toggle — hide part of a sentence or paragraph behind a "Show
More" button.

**Key options:** Visible Content, Hidden Content, Button Text (Show/Hide), button locations,
Merge Boundary Paragraphs, Show Ellipsis When Collapsed, Word/Character Count, Click-Anywhere
to Expand, use native `<details>`, Toggle Icon, Initially Open, button color.

## Widget Area

Render a registered sidebar / widget area inside the page.

**Key options:** Sidebar.
