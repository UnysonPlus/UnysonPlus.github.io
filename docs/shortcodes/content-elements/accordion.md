# Accordion

Collapsible accordion panels.

## Options

Accordion Items (Title, Content, Open by Default), Title Tag, Icon Style & Position, custom open/closed-state image or text, Item Numbering, Start Number, Item Spacing, Title Alignment, Collapsible, Multiple Open, URL Hash Deep-Linking, Show Expand/Collapse All buttons, and per-state title/content/icon colors.

> Shared with most elements: **Background Color**, **Text Color**, and **Margin & Padding**.
