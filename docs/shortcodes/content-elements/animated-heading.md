# Animated Heading

A heading where part of the text rotates through several words with a typewriter, fade, slide,
flip, zoom or clip animation. Tabs: **Content**, **Design**, **Styling**, **Animations**,
**Advanced**.

## Content

| Option | Choices |
| --- | --- |
| **Before Text** | Static text before the rotating words |
| **Rotating Words** | One word/phrase per line |
| **After Text** | Static text after the rotating words |
| **HTML Tag** | H1–H6 · Paragraph · Div |

## Design

| Option | Choices |
| --- | --- |
| **Animation** | Typewriter · Fade · Slide · Flip · Zoom · Clip |
| **Speed** | Slow · Normal · Fast |
| **Word Highlight** | None · Accent color · Underline · Marker background |
| **Alignment** | Left · Center · Right |

## Styling

**Text Color**, **Highlight / Accent Color**, a **Font Size Preset**, and **Margin & Padding**.
