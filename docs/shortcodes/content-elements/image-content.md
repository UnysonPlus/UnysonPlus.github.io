# Image Content

An image alongside text in a responsive two-column layout.

## Options

Image, Content, Image Alt Text, Image Link, Open in New Window, Layout, Column Ratio, Vertical Alignment, Gap, Mobile Stacking Order, Image Fit, Image Border Radius, Image Shadow.

> Shared with most elements: **Background Color**, **Text Color**, and **Margin & Padding**.
