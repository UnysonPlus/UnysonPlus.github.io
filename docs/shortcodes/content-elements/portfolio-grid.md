# Portfolio Grid

A filterable grid of portfolio projects with category filter buttons. Provided by the
**Portfolio** extension — activate it from **Unyson+ → Extensions** for this element to appear
in the builder. Tabs: **Content**, **Layout**, **Styling**, **Animations**, **Advanced**.

## Content

![Portfolio Grid options panel — Content tab](/img/shortcodes/portfolio-grid-content.png)

| Option | Choices |
| --- | --- |
| **Categories** | Limit to these portfolio categories (empty = all) |
| **Number of projects** | Default `-1` (all) |
| **Featured only** | Show only featured projects |
| **Order by** | Date · Custom order · Title · Random |
| **Order direction** | Descending · Ascending |

## Layout

![Portfolio Grid options panel — Layout tab](/img/shortcodes/portfolio-grid-layout.png)

| Option | Choices |
| --- | --- |
| **Columns (Desktop)** | 1–6 |
| **Gap** | Pixels (default `24`) |
| **Thumbnail Image Size** | Thumbnail · Medium · Medium Large · Large · Full |
| **Category filters** | Show filter buttons |
| **Show summary** | Display project summaries |

## Styling

![Portfolio Grid options panel — Styling tab](/img/shortcodes/portfolio-grid-styling.png)

**Text Color**, **Background Color**, a **Font Size Preset**, and **Margin & Padding**.
