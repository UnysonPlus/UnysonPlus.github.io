# Special Heading

An overline + title + subtitle heading block.

## Options

Overline, Title, Subtitle, Title Tag, per-part Alignment, Overline Uppercase, Overline Marker & Position, Element Spacing, Heading Max Width, Title Display Size, Subtitle Font Size/Max Width, per-part colors and CSS classes.

> Shared with most elements: **Background Color**, **Text Color**, and **Margin & Padding**.
