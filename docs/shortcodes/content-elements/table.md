# Table

A data table with optional interactivity.

## Options

Table Preset, Frame (Border Preset), Alternating Row Colors, Row Hover Highlight, Bordered, Compact, Sticky Header, Caption & Position, Sorting, Search/Filter, Pagination, Rows Per Page, Rows-Per-Page Selector, Info Line.

> Shared with most elements: **Background Color**, **Text Color**, and **Margin & Padding**.
