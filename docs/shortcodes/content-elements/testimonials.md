# Testimonials

Display testimonials in a **grid** or **carousel**.

## Layout Type

Choose **Grid** or **Carousel**. Grid uses responsive columns; Carousel adds autoplay,
arrows, indicators, and loop controls.

:::note Screenshots — grid vs carousel
Add `testimonials-grid` and `testimonials-carousel` screenshots to `static/img/shortcodes/`.
:::

## Items

Each testimonial: Quote, Image (avatar), Name, Position, Website Name, Website Link, Rating.

## Layout & cards

Layout Type, Grid Columns (Desktop), Gutter Size, Text Align, Container, Items per Slide
(Carousel), Card Style, Avatar Position / Shape / Size, Show Rating Stars.

## Carousel controls

Autoplay, Autoplay Interval (ms), Pause on Hover, Show Prev/Next, Show Indicators, Indicator
Style, Wrap Around (loop).

## Styling

Section Title, plus colors for Quote, Author Name, Author Job, Site Link, Background, and
Margin & Padding.
