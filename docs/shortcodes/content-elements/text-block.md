# Text Block

A rich-text (WYSIWYG) block. Tabs: **Content**, **Styling**, **Animations**, **Advanced**.

## Content

| Option | Notes |
| --- | --- |
| **Content** | Rich text; supports formatting and nested shortcodes |

## Styling

Text Color, Background Color, and Margin & Padding.
