# Widget Area

Render a registered sidebar / widget area inside the page. Tabs: **Content**, **Styling**,
**Animations**, **Advanced**.

## Content

| Option | Notes |
| --- | --- |
| **Sidebar** | Select which registered sidebar / widget area to render |

## Styling

Text Color, Background Color, and Margin & Padding.
