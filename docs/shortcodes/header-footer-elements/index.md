# Header / Footer Elements

Site-chrome elements used mainly inside the [Theme Builder](/docs/extensions/theme-builder),
though they work anywhere on the page canvas. They pull live data from WordPress — menus,
Site Identity, Theme Settings social profiles — so a header you build stays in sync with the
site's own settings.

- **[Navigation Menu](/docs/shortcodes/header-footer-elements/nav-menu)** — display a WordPress
  menu, horizontal or vertical, with dropdown / mega / accordion submenus.
- **[Site Logo](/docs/shortcodes/header-footer-elements/site-logo)** — the logo or title from
  Site Identity, or a custom image.
- **[Site Search](/docs/shortcodes/header-footer-elements/site-search)** — an inline or
  click-to-expand search form.
- **[Social Icons](/docs/shortcodes/header-footer-elements/social-icons)** — a row of
  social-profile links, from Theme Settings or a manual list.
- **[Menu Toggle](/docs/shortcodes/header-footer-elements/menu-toggle)** — a hamburger button
  that opens an off-canvas drawer.

**Cart** — a cart icon with a live item-count badge — is part of the
**[WooCommerce extension](/docs/extensions/woocommerce)** and appears only when WooCommerce is
active.

> Most elements also expose an **Advanced** tab (visibility, custom class/ID, spacing).
