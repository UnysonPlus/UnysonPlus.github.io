# Media Elements

Drop-in image and video elements, available from the **Media Elements** tab of the Page
Builder.

> Both elements also expose **Background Color** and **Margin & Padding**.

## Image

<img src="/img/shortcodes/image-backend.png" alt="Image on the Page Builder canvas" width="936" />

A single image with sizing and an optional link.

**Key options:** Choose Image, Width, Height, Image Link, Open Link in New Window.

## Video

An embedded video from a URL (YouTube, Vimeo, self-hosted, etc.).

**Key options:** Insert Video URL, Video Max Width, Aspect Ratio.

## Before / After

An interactive before/after image comparison slider — drag, hover or click to reveal — in many
handle styles, horizontal or vertical.

**Key options:** Before/After Image, Show Labels (+ Before/After Label), Design (classic,
labeled, framed, arrow, circle…), Orientation (Horizontal / Vertical), Interaction (Drag /
Follow cursor / Click to toggle), Start Position (0–100%), Auto Intro Sweep, Image Ratio, Max
Width, Corner Radius, Handle Size, and per-part colors (Divider, Handle, Handle Icon, Label
Background/Text).

## Image Box

<img src="/img/shortcodes/image-box-backend.png" alt="Image Box on the Page Builder canvas" width="936" />

An image paired with a heading, text and link, in many hover-overlay, caption, card and frame
designs. Tabs: **Content**, **Design**, **Effects & Link**, **Styling**.

**Key options:** Image (+ Alt override), Eyebrow/Subtitle, Title (+ HTML tag), Text, Icon (or
Custom emoji/SVG), Button/Link Style (None / Button / Text link / Arrow link); Design (stacked,
overlay, caption, side-left/right, frame, feature…), Image Crop Ratio, Media Width (side
designs), Content Alignment; Hover Effect (zoom in/out, grayscale→color, blur, shine, lift, 3D
tilt), Transition Speed, Overlay Color/Opacity, Link Behavior (URL / image lightbox / video
lightbox); plus Title/Eyebrow/Text/Icon/Accent colors.

## Image Hotspots

An image with interactive pins; each pin reveals a tooltip card with a title, text and link.

**Key options:** Image, Hotspots (repeatable: X/Y position 0–100%, Pin Icon, Title, Text,
Link Label/URL, Open in New Tab); Pin Style (pulse, dot, circle, icon, plus, asterisk…), Open
Tooltip On (Hover / Click), Pin Size, Image Corner Radius; plus Pin / Tooltip Background /
Tooltip Text / Link colors.

## Video Popup

<img src="/img/shortcodes/video-popup-backend.png" alt="Video Popup on the Page Builder canvas" width="936" />

A poster image with a play button that opens a YouTube / Vimeo / self-hosted video in a
lightbox.

**Key options:** Poster Image, Video URL, Play Label, Caption / Accessible Label; Play Button
Style (classic, minimal, modern, circle, gradient…), Poster Ratio (16:9, 4:3, 1:1, 21:9), Play
Button Size, Corner Radius, Darken Poster, Zoom Poster on Hover; plus Play Button / Play Icon /
Overlay / Label colors.
