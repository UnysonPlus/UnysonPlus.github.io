# Media Elements

Drop-in image and video elements, available from the **Media Elements** tab of the Page
Builder.

> Both elements also expose **Background Color** and **Margin & Padding**.

## Image

A single image with sizing and an optional link.

**Key options:** Choose Image, Width, Height, Image Link, Open Link in New Window.

## Video

An embedded video from a URL (YouTube, Vimeo, self-hosted, etc.).

**Key options:** Insert Video URL, Video Max Width, Aspect Ratio.
