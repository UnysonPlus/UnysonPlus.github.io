# Media Elements

Image, video and visual elements for the page canvas — most are available from the
**Media Elements** tab of the Page Builder.

- **[Image](/docs/shortcodes/media-elements/media-image)** — a single image with sizing and an
  optional link.
- **[Video](/docs/shortcodes/media-elements/media-video)** — an embedded video from a URL
  (YouTube, Vimeo, self-hosted).
- **[Gallery](/docs/shortcodes/media-elements/gallery)** — a multi-image gallery.
- **[Image Box](/docs/shortcodes/media-elements/image-box)** — an image paired with heading,
  text and link, in many hover / overlay / caption / card designs.
- **[Image Content](/docs/shortcodes/media-elements/image-content)** — an image beside a block
  of content.
- **[Before / After](/docs/shortcodes/media-elements/before-after)** — an interactive
  comparison slider.
- **[Image Hotspots](/docs/shortcodes/media-elements/image-hotspots)** — an image with
  interactive pins and tooltip cards.
- **[Video Popup](/docs/shortcodes/media-elements/video-popup)** — a poster with a play button
  that opens a video lightbox.
- **[Audio Player](/docs/shortcodes/media-elements/audio-player)** — a single track or playlist
  with cover art, in several designs.
- **[Lottie Animation](/docs/shortcodes/media-elements/lottie)** — plays a Lottie `.json`
  vector with autoplay, hover, on-scroll or click triggers.
- **[Map](/docs/shortcodes/media-elements/map)** — an embedded map.
- **[Icon](/docs/shortcodes/media-elements/icon)** — a single vector icon.
- **[Logo Grid](/docs/shortcodes/media-elements/logo-grid)** — a grid, carousel or marquee of
  client / partner logos.
- **[Portfolio Grid](/docs/shortcodes/media-elements/portfolio-grid)** — a filterable
  portfolio grid.
- **[Project Gallery](/docs/shortcodes/media-elements/project-gallery)** — a project image
  gallery.

> Most image / video elements also expose **Background Color** and **Margin & Padding**.
