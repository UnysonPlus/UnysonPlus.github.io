# Page Builder Elements

The Page Builder ships with **27 built-in elements** (shortcodes). You add them by dragging
them onto the page canvas and configuring them through their options panel — no code
required. They're grouped into three tabs in the builder:

- **[Content Elements](/docs/shortcodes/content-elements)** — text, buttons, media boxes, posts, tables, and more (20 elements, each with its own page).
- **[Layout Elements](./layout-elements.md)** — sections, rows, columns and special section types (5 elements).
- **[Media Elements](./media-elements.md)** — image and video (2 elements).

## All elements at a glance

| Element | Category | What it does |
| --- | --- | --- |
| Accordion | Content | Collapsible accordion items |
| Button | Content | A styled, linkable button |
| Calendar | Content | An events/date calendar |
| Call To Action | Content | A title + content + button CTA block |
| Code Block | Content | Raw HTML/CSS/JavaScript block |
| Divider | Content | A horizontal rule / separator, optionally with text or icon |
| Icon | Content | A single icon with optional title |
| Icon Box | Content | An icon paired with title and content |
| Image Content | Content | Image alongside text in a responsive two-column layout |
| Map | Content | An interactive map (multiple engines) |
| Notification | Content | A dismissible notification / alert box |
| Posts | Content | Query and display posts in grid/list/masonry/slider/overlay/hero layouts |
| Special Heading | Content | An overline + title + subtitle heading block |
| Table | Content | A data table with sorting, search and pagination |
| Tabs | Content | Tabbed content panels |
| Team Member | Content | A team member card (photo, name, role) |
| Testimonials | Content | Testimonials in grid or carousel |
| Text Block | Content | A rich-text (WYSIWYG) block |
| Text Expander | Content | Inline read-more / reveal toggle |
| Widget Area | Content | Render a registered sidebar / widget area |
| Section | Layout | A full content section with background options |
| Hero Section | Layout | A section with a parallax background image |
| Masonry Section | Layout | A section that packs columns into a masonry grid |
| Row | Layout | A builder row (column container) |
| Column | Layout | A responsive column with width/offset/order controls |
| Image | Media | A single image with sizing and link |
| Video | Media | An embedded video (URL based) |

Common options shared by most elements include **Background Color**, **Text Color**, and
**Margin &amp; Padding** for spacing control.
